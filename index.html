What this is

I prompted 5 AI models on every 2026 FIFA World Cup knockout stage match — asking each to predict the winner, scoreline, confidence level, and reasoning before the game was played. No search, no live data. Just what the models know.

The goal was to see how well LLMs actually reason under uncertainty — and whether their confidence scores mean anything.

What I collected per prediction

Exact scoreline, including extra time or penalties if the model predicted them

Self-reported — 1 = pure guess, 10 = very confident

Reasoning

One sentence from the model explaining its pick

Methodology

No search: every model was run without web search or live data. Not all models support search, so turning it off kept things fair.

Fresh context every time: each prediction was made in a brand new conversation — new chat, new account, no history carried over. This keeps every run fully independent with no cross-contamination between predictions.

Timing: prompts were run after the matchup was confirmed (following the previous round) but before the game kicked off. No hindsight, no live data.

One prompt per game for the leaderboard: each model was prompted once per game for the leaderboard. That’s what the leaderboard and per-game tabs are based on.

Consistency study: the semifinals and Final were each run 5 times per model in separate fresh conversations, using the identical prompt. The goal was to see whether models give stable answers or flip — and whether their confidence tracks with it. That data is a separate reliability layer on top of the prediction study.

Scoring: correct winner (yes/no) and exact score (yes/no), checked against the actual result after each game.
